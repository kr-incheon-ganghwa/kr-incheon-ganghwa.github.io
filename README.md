# 강화군 · 인천광역시

> 강화군 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-incheon-ganghwa.github.io/**

강화군은 인천광역시 북서부에 자리한 도서(島嶼) 지역으로, 본섬인 강화도를 비롯해 석모도·교동도·볼음도·주문도 등 유인도와 무인도를 아우른다. 면적은 약 411.4㎢로 인천광역시 전체 면적의 약 40%를 차지하는 최대 기초자치단체이며, 행정구역은 1읍 12면(강화읍과 선원·불은·길상·화도·양도·내가·하점·양사·송해·교동·삼산·서도면)으로 구성된다. 한강·임진강·예성강 하구가 만나는 지점에 위치해 북쪽으로는 북한 개풍·연백 지역과 바다를 사이에 두고 마주하는 접경지대다. 청동기 시대 고인돌부터 고려의 임시 수도, 조선의 국방 요충지, 개항기 외세와의 충돌 현장까지 한반도 역사의 굵직한 장면이 이 섬에 응축돼 있어 '지붕 없는 박물관'으로 불린다. 마니산 참성단, 부근리 고인돌, 고려궁지, 광성보·초지진 등 유적이 곳곳에 남아 있으며, 순무·인삼·화문석 같은 특산물과 넓은 갯벌 생태계로도 이름 높다.
